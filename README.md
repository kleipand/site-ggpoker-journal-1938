# site-ggpoker-journal-1938

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存储，便于浏览和回溯。仓库本身不针对任何特定域名或网站，仅作为文档和页面资源的集合。

## 目录说明

```
site-ggpoker-journal-1938/
├── index.html          # 入口页面（如有）
├── pages/              # 存放主要 HTML 页面
├── assets/             # 样式、脚本、图片等资源文件
└── archive/            # 归档旧版本或历史页面
```

- `pages/`：存放独立发布的 HTML 文件，每个文件可视为一个独立的文档或页面。
- `assets/`：用于存放页面引用的公共资源，如 CSS、JavaScript、图片等。
- `archive/`：用于存放不再活跃但仍需保留的历史版本。

## 页面归档说明

本仓库的页面内容可能随时间更新或补充。归档目录中的文件为历史快照，保留原始状态以供参考。主目录下的页面为当前可用版本。

- 归档文件命名建议包含日期或版本标识。
- 不保证所有页面在所有环境下均完全兼容，但尽量遵循基本 Web 标准。
